Termin zdania nie moze zostac przedluzony

Termin zwrotu ksiazki, ktorz starasz się przedluzyc, nie moze zostac przedluzony. Mozliwe przyczyny to:

- Osiagnieto maksymalny okres uzytkowania.
- Termin zwrotu jest już przedluzony o 3 tygodnie.
- Ktos inny zarezerwowal ksiazke.

Prosimy pamietac o terminowym zwroceniu tej ksiazki do biblioteki.
